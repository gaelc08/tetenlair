# Têtes en l'Air & Compagnie

Site web statique — [tetesenlairetcie.com](https://tetesenlairetcie.com)

## Description

Site vitrine pour **Têtes en l'Air & Compagnie**, troupe de spectacles déambulatoires basée à **Neufmanil, Ardennes**.

La compagnie propose des spectacles thématiques pour événements publics et privés — festivals, fêtes de ville, inaugurations, cavalcades, marchés de Noël.

## 🎪 Spectacles

| Spectacle | Thème |
|---|---|
| 🔴⚪ **Rouge & Blanc** | Élégance, échassiers haut de gamme |
| ⚙️ **Steampunk** | Angleterre victorienne, engrenages, cracheurs de flammes |
| 🧝 **Médiéval Fantastique** | Lutins, elfes, farfadets |
| 🌸❄️☀️🍂 **Les 4 Saisons** | Poétique et familial |
| ⚪ **Paradis Blanc** | Féerie hivernale, marchés de Noël |

## 🛠️ Stack technique

- **HTML/CSS/JS** statique pur
- **Génération :** Scripts Python (`build_pages.py`) — non versionnés dans ce repo
- **Polices :** Playfair Display + Raleway (Google Fonts)
- **Galeries photo :** [GLightbox](https://biati-digital.github.io/glightbox/)
- **Images :** Hébergées sur [Cloudinary](https://cloudinary.com/)
- **Déploiement :** Netlify
